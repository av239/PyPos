# PyPos
Port scanner similar to nmap in Python
## Usage

usage: PyPos.py [-h] -host HOST [-start START] [-end END]

PyPoS v0.2

optional arguments:
  **-h**, **--help**    show this help message and exit
  
  **-host** HOST    Host to scan
  
  **-start** START  Starting port
  
  **-end** END      Ending port
